# ImageCarousel - 本地漫畫/圖片自動輪播器

一個輕量級、零安裝、純靜態的網頁版圖片瀏覽器，專為閱讀本機端的漫畫或圖片集設計。

## 專案特色

*   **零安裝 (Zero Install)**: 不需要 Node.js、Python 或任何後端伺服器，直接用瀏覽器開啟 `index.html` 即可使用。
*   **隱私安全**: 所有圖片讀取與處理皆在您的**本地端**完成，不會上傳至任何雲端伺服器。
*   **高效能**: 採用 Lazy Loading 與非同步解碼技術，即使是數百張高解析度圖片也能流暢瀏覽。
*   **記憶功能**: 自動記住上次瀏覽的資料夾與偏好設定 (輪播速度、顯示模式)。

## 功能列表

1.  **自動輪播**: 支援自訂秒數自動翻頁 (1s, 3s, 5s...等)，並可由設定檔擴充選項。
2.  **多種顯示模式**:
    *   **Fit**: 完整顯示 (保持比例)
    *   **Fill**: 填滿視窗 (裁切邊緣)
    *   **1:1**: 原始大小 (可捲動)
3.  **沉浸式閱讀**:
    *   支援 **全螢幕** 模式 (Toggle Fullscreen)，自動隱藏介面，僅在滑鼠移動到底部時顯示。
    *   可手動收合下方控制面板 (▼ 按鈕)，最大化閱讀空間。
4.  **直覺導航**:
    *   支援鍵盤方向鍵 (`←`/`→`) 翻頁。
    *   支援點擊螢幕左右兩側 (10% 區域) 快速翻頁。
    *   下方縮圖列表 (Thumbnail Strip) 快速預覽跳轉。
5.  **圖庫管理**:
    *   支援讀取 `ImageDB` 下的多個子資料夾。
    *   透過下拉選單快速在不同漫畫/圖庫間切換。
    *   智慧排序 (支援 1.jpg, 2.jpg, 10.jpg 自然排序)。

## 使用操作說明

### 1. 準備檔案
請建立一個資料夾 (例如 `ImageCarousel`)，並確認結構如下：
```
ImageCarousel/
├── index.html        (主程式)
├── js/               (程式邏輯)
├── css/              (樣式表)
└── 您的漫畫資料夾/    (名稱自訂)
    ├── 漫畫A/
    │   ├── 01.jpg
    │   ├── 02.jpg
    │   └── ...
    └── 漫畫B/
        └── ...
```
> **注意**: 您可以自由命名資料夾，不一定要叫 `ImageDB`。

### 2. 啟動程式
1.  直接雙擊 `index.html`，建議使用現代瀏覽器 (Chrome, Edge) 開啟以獲得最佳效能。
2.  點擊畫面中央的 **「載入資料夾」** 按鈕。
3.  選擇您的圖片資料夾。
4.  瀏覽器上方會跳出權限詢問視窗，請點選 **「允許」** 或 **「檢視檔案」**。
    *   *下次開啟時，可直接點擊「快速載入上次的資料夾」跳過選擇步驟。*

### 3. 操作方式

| 動作 | 滑鼠 / 觸控 | 鍵盤快捷鍵 |
| :--- | :--- | :--- |
| **上一張** | 點擊畫面左側 (10% 區域) 或 控制列 `⏮` | `←` (左方向鍵) |
| **下一張** | 點擊畫面右側 (10% 區域) 或 控制列 `⏭` | `→` (右方向鍵) |
| **暫停/播放** | 點擊控制列 Play 按鈕 | `Space` (空白鍵) |
| **全螢幕** | 點擊控制列 `Toggle Fullscreen` | `Esc` (退出) |
| **顯示模式** | 點擊 `⛶ Fit/Fill/1:1` 切換 | - |
| **收合介面** | 點擊右下角 `▼` 按鈕 | - |

## 技術細節
*   **核心技術**: HTML5, CSS3, Vanilla JavaScript (無框架)。
*   **檔案存取**: 使用 **File System Access API** 直接讀取本地資料夾。
*   **狀態保存**: 使用 **IndexedDB** 儲存資料夾權限 Handles，使用 **localStorage** 儲存 UI 設定。
*   **效能優化**: 使用 **IntersectionObserver** 與 **createImageBitmap** 實作縮圖懶加載與快速解碼。
